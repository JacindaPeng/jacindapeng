# 娃哈哈 AD 钙奶 H5 互动广告

作品主题：`AD钙奶30周年：不管几岁 快乐万岁`

## 文件说明

- `index.html`：移动端 H5 主页面
- `styles.css`：视觉样式与动效
- `script.js`：选择、游戏、结果卡、分享逻辑
- `assets/`：已压缩的产品图素材

## 本地预览

直接双击 `index.html`，或在本目录启动静态服务：

```bash
python -m http.server 8080
```

打开 `http://localhost:8080` 预览。

## 提交建议

将 `h5-ad` 文件夹部署到可公开访问且一年内有效的静态站点，再把发布链接生成二维码，放入 Word 文档提交给学校。
