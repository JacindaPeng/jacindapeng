# SANFU PC Web Prototype（内容优先 · 甜酷潮流）

这是一个无需构建工具的静态网页交互原型（HTML/CSS/JS），用于展示：
- 首页（社区封面 + 灵感流）
- 搭配流 / 搭配详情（一键成套）
- 分类列表（风格入口 + 对比抽屉）
- 商品详情（内容化 PDP）
- 门店（地图 + 列表 + 到店打卡指引）

## 运行方式
1. 打开 `pc-web-prototype/index.html`（双击即可，或拖拽到浏览器）
2. 推荐使用 Chrome / Edge

## 说明
- 数据为 mock，用于演示交互与信息架构
- 状态会写入 localStorage：收藏、购物袋、对比、门店定位等
